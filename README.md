## 서울 지하철 요금
### 지하철 요금
**교통카드**
	기본운임(10km 까지)
		일반 : 1,250원
		청소년 : 720원
		어린이 : 450원
	추가운임
		10km초과 50km이하 : 5km 마다 100원
		50km초과 : 8km마다 100원
**1회용 교통카드**
	교통카드 운임에 100원 추가
		일반(청소년 포함) : 1,350원
		어린이 : 450원
